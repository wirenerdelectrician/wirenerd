Build Guide · Motor Control

Start/Stop with Seal-In

The cornerstone circuit. Momentary Start and Stop buttons with a seal-in contact that holds the motor running after you release Start.

Difficulty: CoreBuilds on: two-wire controlLesson: read the theory →

What you’re building

Parts to add

  • 1Power Source
  • 1Push Button NC (Stop)
  • 1Push Button NO (Start)
  • 1Coil
  • 1NO Contact
  • 1Contactor
  • 1Motor

Steps

  1. Add and label

    Add the parts. Assign the Coil the letter M. Assign the NO Contact the letter M (this is the seal-in). Assign the Contactor M.

  2. Wire the main control rung

    Stop (NC) then Start (NO) in series to the coil.

    Power Source (L1) → Stop button
    Stop button → Start button
    Start button → Coil A1
    Coil A2 → Power Source (N)

  3. Add the seal-in around Start

    Wire the M NO Contact in PARALLEL with the Start button — across the same two points Start connects.

    Stop→Start junction → M NO Contact
    M NO Contact → Start→Coil junction

  4. Wire the motor power

    Power Source (L1) → Contactor → Motor → Power Source (N)

  5. Energize and test

    Tap Start. The motor runs and KEEPS running after you release — the seal-in holds it.

✓ Test it

  • Tap Start, release: motor stays running (seal-in working).
  • Tap Stop: motor drops out and stays off.
  • After Stop, nothing restarts on its own — that is the safety of three-wire control.
Tip: If the motor stops the instant you release Start, your seal-in contact is not wired in parallel with Start, or it is not assigned the coil letter M.
