Build Guide · Motor Control

Jogging (Inch Control)

Run a motor only while a button is held, by deliberately defeating the seal-in so each press is a momentary burst.

Difficulty: AdvancedBuilds on: start/stop with seal-inLesson: read the theory →

What you’re building

Parts to add

  • 1Power Source
  • 1Push Button NC (Stop)
  • 1Push Button NO (Start)
  • 1Push Button two-block NO+NC (Jog)
  • 1Coil
  • 1NO Contact (seal-in)
  • 1Contactor
  • 1Motor

Steps

  1. Start from a working start/stop

    Build the normal Start/Stop/seal-in circuit first (Coil M, seal-in NO Contact M, Contactor M). Get it sealing in.

  2. Add the Jog button’s NO block

    Wire the jog button’s NO block to energize the coil — a parallel path to Start.

    Stop → Jog NO block → Coil A1

  3. Route the seal-in through the Jog NC block

    Put the jog button’s NC block in series with the seal-in contact, so pressing Jog opens the holding path.

    seal-in M NO Contact → Jog NC block → (continue to coil)

  4. Energize and compare

    Try Start (latches and runs) vs. Jog (runs only while held).

✓ Test it

  • Press Start: motor latches and keeps running.
  • Press and HOLD Jog: motor runs; release: motor stops immediately.
  • Jog never seals in — its NC block kills the holding path while you press it.
Tip: Hold the Jog button down (mousedown) to keep it active; release to spring it back, just like a real momentary button.
